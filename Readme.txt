wav2mono

■ 使い方
WAVファイルをこのソフトのウィンドウにドラッグ＆ドロップするだけで、モノラルに変換されます。
変換後のファイルは、モノラルとデュアルモノのファイルはmonoフォルダに、ステレオのファイルはstereoフォルダに保存され、3ch以上のファイルはmultichannelフォルダに保存されます。
